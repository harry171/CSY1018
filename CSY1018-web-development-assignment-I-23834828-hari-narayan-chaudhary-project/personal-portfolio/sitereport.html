site report

The assignment was to develop portfolio website using html and CSS. This assignment is based on web development. We have used VS code as code editor for this project. This website shows the professional background of and individual. It displays brief information about work experience skills and contact details along with social media links.
Introduction:
Personal portfolio website provides brief information about the professional background of and individual. This project consists of two files. Html is used to create markup of the website and then CSS is used to provide more style to the website.
This project consists of 5 webpages which are listed below:
1.index.html:
This section displays name, current work status and where I am from. On the bottom it also has navigation bar which displays other tabs of the website.
2.workexperience.html:
This section displays my personal information and my work experience in the web development field. It also shows my skills and coding languages I have mastered.
3.portfolio.html:
In this section sample portfolio images are displayed of my completed projects and works.
4.sitereport.html:
In this section I have included my full report of developing this website from the scratch.
5.contact.html:
In this webpage I have included my contact information as well as message section from where people can message me whenever they want.
Software used:
1.VS code
2.Html
3.CSS
Significance:
Personal portfolio presents work and all the basic information of an individual. Personal portfolio allows others to get understanding of your work, passion and your skills.
